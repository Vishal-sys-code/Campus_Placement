#include <iostream>
int main() {
  int value1{ 5 };
  int value2{ 6 };
  int &ref{ value1 }; 
  std::cout<<ref<<" ";
  ref = value2; 
  std::cout<<ref<<" ";
  ref = value2; 
  std::cout<<ref<<" ";
}

Output:
5 6 6 


Generally, the references that we have covered till now are called non-const references. 
Because, these references were aliases for non-const values.
